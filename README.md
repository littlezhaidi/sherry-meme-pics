# 橘雪莉生成器 (Node.js 版)

这是一个基于 Node.js 和 Express 的 Web 应用程序，复刻了原 Python 版的橘雪莉生成器功能。它允许用户在网页上选择背景图片、输入文字，并生成带有自定义字体样式的橘雪莉表情包/图片。

## ✨ 功能特点

- **Web 可视化界面**：简洁的网页操作界面。
- **背景选择**：内置多种橘雪莉背景图片供选择。
- **文字自定义**：
  - 支持多行文本输入。
  - 自动换行排版。
  - **字体选择**：支持多种中文字体。
  - **样式调整**：可调节字体大小、文字颜色。
  - **特效支持**：支持文字加粗、描边（可自定义描边宽度）。
- **图片生成**：服务端生成高清图片并返回前端展示。

## 🛠️ 安装说明

在使用本程序之前，请确保您的系统中已安装 [Node.js](https://nodejs.org/)。

1. **进入项目目录**
   ```bash
   cd node-app
   ```

2. **安装依赖**
   ```bash
   npm install
   ```
   *注意：本项目依赖 `canvas` 库，安装过程中可能需要编译环境（如 Windows 上的构建工具）。如果遇到安装错误，请参考 `node-canvas` 的官方安装文档。*

## 🚀 使用方法

1. **启动服务器**
   ```bash
   node server.js
   ```
   当看到终端输出 `Server is running on http://localhost:3000` 时，表示启动成功。

2. **访问应用**
   打开浏览器，访问：[http://localhost:3000](http://localhost:3000)

3. **生成图片**
   - 在左侧/上方控制面板选择背景图。
   - 输入想要显示的文字。
   - 调整字体、颜色、大小等参数。
   - 点击“生成图片”按钮，右侧/下方将显示生成结果。

## 📂 项目结构

```
node-app/
├── background_images/  # 背景图片素材目录
├── Font/              # 字体文件目录
├── public/            # 前端静态资源
│   ├── index.html     # 主页
│   ├── style.css      # 样式表
│   └── script.js      # 前端交互逻辑
├── output_images/     # 生成图片的临时输出目录（自动创建）
├── server.js          # Node.js 后端服务器入口
├── package.json       # 项目配置与依赖管理
└── README.md          # 项目说明文档
```

## ⚠️ 注意事项

- **字体文件**：请确保 `Font` 目录下包含所需的 `.ttf`, `.otf` 或 `.ttc` 字体文件。
- **背景图片**：请确保 `background_images` 目录下包含 `.png`, `.jpg` 或 `.jpeg` 格式的背景图。
- **中文路径**：程序已针对 Windows 下的中文路径/文件名进行了处理，可正常读取中文命名的资源文件。
